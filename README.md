<h2> Project </h2>
We started this project for fun and to fix our knowlodge on tools and technologies like Quarkus and Reactive Native. This project consists in an app that creates a voting poll for songs and playlist and play the most voted ones on Spotify.  

Our main goal is to make the app containerized and fully scalable on Kubernetes or any other container orchestration tools.
